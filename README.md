# AeroVista-Premium-Service-Companion
AeroVista Premium Service Companion is a personalized inflight service experience available through the AeroVista mobile app and seatback screen for premium-cabin passengers. It allows travelers to set meal timing, beverage preferences, rest preferences, and special occasion details before or during the flight.
